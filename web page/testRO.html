Functia perfect primeste ca parametru un intreg n si intoarce 1 daca este perfect, respectiv 0 in caz contrar. Cititi un intreg si verificati daca este perfect afisand mesaje corespunzatoare.


Functia esteInVector are ca parametri un vector de maxim 10000 intregi, numarul exact de elemente n, un intreg oarecare x si intoarce numarul de aparitii ale lui x in vector. Cititi un vector si afisati de cate ori apare primul element in acel vector.


Functia vectorSortat primeste ca parametri un vector de maxim 1000 intregi, numarul exact de elemente si intoarce 1 daca vectorul este sortat crescator, respectiv 0 in caz contrar. Cititi o matrice patratica de maxim 100*100 intregi si afisati numarul de linii sortate crescator.


Functia inter primeste ca parametri doi intregi x si y si interschimba valorile parametrilor. Cititi un vector de maxim 1000 intregi si interschimbati elementele situate consecutiv care au proprietatea primul element din pereche mai mic de cat al doilea. Afisati vectorul obtinut.


Realizati o functie cifreComune care primeste ca parametri doi intregi a, b si intoarce numarul de cifre comune celor doua numere.
